# Social Media Clone

A full-stack social media clone built with **React**, **Tailwind CSS**, **Express**, **Express-Session**, and **SQLite3**.  
It allows users to sign up, sign in, create posts, like posts, comment, and edit their profile â€” providing the basic features of a social media platform.

---

## Features

- User authentication with hashed passwords
- Create, read, update posts
- Like posts
- Comment on posts
- Edit user profile and bio
- Real-time success/error messages
- Input validation and sanitization for security

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, React Router
- **Backend:** Node.js, Express, Express-Session
- **Database:** SQLite3
- **Security:** Password hashing (bcrypt), input validation, session-based authentication

---

## Getting Started

### Backend

1. Go to the backend folder:
cd backend

2. Start server:
npm start

### Frontend
1. Go to the frontend folder:
cd frontend

2. Start app:
npm run dev

Open your browser at http://localhost:5173 (or the port Vite shows).
